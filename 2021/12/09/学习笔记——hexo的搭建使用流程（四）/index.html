一、在hexo博客中添加图片

1、安装插件

npm install hexo-asset-image --save

2、修改配置文件

修改主目录下的_config.yml，修改post_asset_folder，把false改成true

3、更改/home/hexo/node_modules/hexo-asset-image/index.js

将该配置文件下的内容进行替换

4、在博客中添加图片

将要插入的图片文件，上传到source/_post/目录下，与博客名字相同的文件夹下

并且在编辑博客内容的时候，按照markdown的标准插入图片
