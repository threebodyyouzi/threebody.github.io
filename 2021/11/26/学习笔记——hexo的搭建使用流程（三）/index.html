一、安装依赖环境

检查主目录下的package.json里是否有 "hexo-helper-live2d": "^3.1.1"这条依赖配置

若没有，使用命令npm install --save hexo-helper-live2d安装依赖环境

二、下载动画model

1、进入动画model的存放目录
